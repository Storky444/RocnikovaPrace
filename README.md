Návod ke spuštění hry

Hru je možné spustit několika způsoby v závislosti na dostupných souborech.

1. Spuštění pomocí Godot Engine
-Otevřete program Godot Engine.
-Klikněte na tlačítko „Import“.
-Vyberte složku s projektem (soubor project.godot).
-Po načtení projektu klikněte na tlačítko „Play“ (F5).
2. Spuštění exportované hry
-Otevřete složku s exportovanou verzí hry.
-Spusťte soubor .exe.
-Hra se automaticky spustí.
3. Ovládání hry
-Pohyb: klávesy W, A, S, D
-Střelba: levé tlačítko myši
-Cíl hry: přežít co nejdéle
